
# LT-Plex

МедиаСервис для оффлайн просмотра всего и чего угодно (в разумных приделах) :3




## Установка для Linux Систем

для начала склонируйте приложение:

```bash
  git clone https://github.com/PythonLTS/LT-Plex.git
  cd LT-Plex

```
чтобы установить :
```bash
  cd installer_Master
  sudo ./Master_Linux
```
после чего запустится установщик на http://localhost:8080
## После установки запуск выглядит так:
если вы не включали автозапуск
```bash
./app/server
```
